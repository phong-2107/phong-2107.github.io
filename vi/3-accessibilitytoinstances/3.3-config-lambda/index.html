Cấu hình cho lambda

Trong bước này chúng ta sẽ cấu hình cho lambda, các kết nối và thiết lập cần thiết

Các bước thực hiện:

  1. Add layer:

    • Mở Lambda fuction

    • trong phần Code Click chọn Layers.

    • Chọn Add a Layer.

    • Chọn Custom layers.

    • Tìm layer-pandas-pyarrow với Version (latest).

    • chọn Add.

  2. cấu hình Environment variables:

    • Chọn Configuration.
    • Tìm Environment variables.
    • chọn Edit.
    • Nhập 2 key: .
    • RAW_BUCKET = edu-logs-raw-<account>.
    • PROCESSED_BUCKET = edu-logs-processed-<account>.
    • chọn Save.

. cấu hình basic settings:

  • Chọn Configuration.
  • Tìm General configuration.
  • chọn Edit.
  • Memory = 2048 .
  • Timeout = 2 min.
  • chọn Save.
  1. Gắn S3 trigger cho Lambda:
    • Tab Configuration → Triggers → Add trigger
      • Source: S3.
      • Bucket: Chọn edu-logs-raw.
      • Event type: chọn All object create events.
      • Prefix: ingest/
      • chọn Add.

  1. Viết mã Lambda Function:
    • Sau khi tạo Lambda function, bạn sẽ cần thêm mã nguồn để xử lý dữ liệu:
import boto3, os, io, json
import pandas as pd
import pyarrow as pa
import pyarrow.parquet as pq
from datetime import timezone
from dateutil import parser as dtparser

s3 = boto3.client("s3")
RAW_BUCKET = os.environ["RAW_BUCKET"]
PROCESSED_BUCKET = os.environ["PROCESSED_BUCKET"]

def _to_df(key: str, body: bytes) -> pd.DataFrame:
    if key.lower().endswith(".json"):
        data = json.loads(body)
        if isinstance(data, dict):
            data = [data]
        return pd.json_normalize(data)
    if key.lower().endswith(".csv"):
        return pd.read_csv(io.BytesIO(body))
    raise ValueError(f"Unsupported format: {key}")

def _norm_cols(df: pd.DataFrame) -> pd.DataFrame:
    df.columns = [c.strip().lower().replace(" ", "_") for c in df.columns]
    # chuẩn hóa timestamp
    for tcol in ("timestamp","event_time","time"):
        if tcol in df.columns:
            df["timestamp"] = pd.to_datetime(df[tcol], errors="coerce", utc=True)
            break
    if "timestamp" not in df.columns:
        df["timestamp"] = pd.Timestamp.utcnow().tz_localize("UTC")
    return df.drop_duplicates()

def lambda_handler(event, _ctx):
    rec = event["Records"][0]
    bucket = rec["s3"]["bucket"]["name"]
    key = rec["s3"]["object"]["key"]

    if bucket != RAW_BUCKET:
        print(f"Skip other bucket: {bucket}")
        return

    obj = s3.get_object(Bucket=bucket, Key=key)
    body = obj["Body"].read()

    df = _norm_cols(_to_df(key, body))
    ts = pd.to_datetime(df["timestamp"].iloc[0], utc=True)
    y, m, d = ts.year, f"{ts.month:02d}", f"{ts.day:02d}"

    table = pa.Table.from_pandas(df)
    buf = io.BytesIO()
    pq.write_table(table, buf, compression="snappy")

    out_key = f"year={y}/month={m}/day={d}/{os.path.splitext(os.path.basename(key))[0]}.parquet"
    s3.put_object(Bucket=PROCESSED_BUCKET, Key=out_key, Body=buf.getvalue())
    print(f"Processed s3://{bucket}/{key} -> s3://{PROCESSED_BUCKET}/{out_key}")
  1. Kiểm tra và deploy Lambda:
    • Sau khi viết mã, click Deploy để triển khai Lambda function.
    • Bạn có thể kiểm tra Lambda bằng cách sử dụng Test trong Lambda Console hoặc thông qua sự kiện S3 mà chúng ta sẽ cấu hình ở bước tiếp theo.
